
## Metodologia

Este projeto utiliza uma abordagem exploratória, com foco em análise descritiva de dados de marketplace, seguindo três etapas principais:

### 1. Coleta de dados
A coleta foi realizada manualmente no Mercado Livre, a partir da busca:
“vela de ignição corsa classic 2014”.

Foram coletados entre 30 e 50 anúncios reais, registrando informações públicas disponíveis na plataforma, como preço, vendas, avaliações, marca, frete e características do anúncio.

### 2. Padronização dos dados
Após a coleta, os dados passam por padronização para garantir comparabilidade entre os anúncios, incluindo:
- normalização de categorias (frete grátis, kit/unidade, loja oficial);
- padronização de avaliações visuais (qualidade da imagem);
- tratamento de campos textuais (marca, descrição completa);
- identificação de dados ausentes.

### 3. Análise exploratória
Com a base padronizada, é realizada análise exploratória para identificar padrões e relações entre variáveis, como:
- preço vs número de vendas;
- frete grátis vs vendas;
- reputação (avaliações) vs vendas;
- marca vs desempenho;
- kit vs unidade.
